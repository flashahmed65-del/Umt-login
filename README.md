# Umt-login

This repository contains a simple static login page (served from `index.html`).

## Live demo (GitHub Pages)

After configuring GitHub Pages for this repository, the site will be available at:

- `https://<your-github-username>.github.io/<repo-name>/`

## How to publish

1. Push this repo to GitHub (e.g., to the `main` branch).
2. In GitHub, go to **Settings > Pages** for this repository.
3. Under **Source**, choose **GitHub Actions** (or select the `main` branch + `/ (root)` if available).
4. Save, then wait a few minutes for the workflow to run and for the site to become available.

> ✅ The workflow file at `.github/workflows/pages.yml` deploys the repository content to GitHub Pages on every push to `main`.
